# CONSULTAS INVENTARIO EXOPLANETARIO

## INTRODUCCIÓN

El trabajo consiste en realizar una página web, a partir de un xml y un xsd que lo valida, con 3 páginas diferentes donde para cada página se haga uso de la base de datos con base x (XQuery) para realizar consultas. 

### CONSULTA 1

He realizado una consulta en la base de datos en la que he filtrado los planetas con temperatura entre 10º y 35º, mostrando en la página web una “galería visual”.

### CONSULTA 2

He realizado una consulta en la base de datos en la que he filtrado los planetas que contienen oxígeno en su atmósfera en un ranking de más cerca a más lejos de la Tierra, mostrando en la página web una “galería visual”.
